# Gleyo

Gleyo is a structured onboarding and retention platform for online communities and Web3 ecosystems.

The platform helps communities improve onboarding, engagement, contributor participation, and long-term retention through quests, automation, wallet integrations, social integrations, and community activity systems.

## Why We Built This

Many online and Web3 communities struggle with:

- poor onboarding
- inactive users after signup
- fragmented engagement systems
- weak retention
- difficulty coordinating contributors across platforms

Gleyo aims to provide a more structured retention and engagement layer for communities.

## Core Features

- Solana wallet connection
- QuickNode / Solana RPC integration
- Community onboarding flows
- Quest and task systems
- User progression tracking
- Discord integration
- Telegram integration
- Twitter/X integration
- Community notifications
- Role and permission systems
- Reward systems
- Contributor management

## Tech Stack

- Python
- Flask
- JavaScript
- HTML/CSS
- Solana Wallet Adapter
- QuickNode RPC
- SQLite/PostgreSQL
